# Downloading Fonts for Refah Mobile

Captain, we need to download the font files. Here's exactly what to do:

## Step 1: Download Cairo Font (for Arabic)

1. Go to: https://fonts.google.com/specimen/Cairo
2. Click "Download family" button (top right)
3. Extract the ZIP file
4. Copy these files to `RifahMobile/assets/fonts/`:
   - `Cairo-Regular.ttf`
   - `Cairo-Medium.ttf`  
   - `Cairo-Bold.ttf`

## Step 2: Download Poppins Font (for English)

1. Go to: https://fonts.google.com/specimen/Poppins
2. Click "Download family" button (top right)
3. Extract the ZIP file
4. Copy these files to `RifahMobile/assets/fonts/`:
   - `Poppins-Regular.ttf`
   - `Poppins-Medium.ttf`
   - `Poppins-Bold.ttf`

## Final Structure:

```
RifahMobile/
└── assets/
    └── fonts/
        ├── Cairo-Regular.ttf
        ├── Cairo-Medium.ttf
        ├── Cairo-Bold.ttf
        ├── Poppins-Regular.ttf
        ├── Poppins-Medium.ttf
        └── Poppins-Bold.ttf
```
